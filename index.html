What are Semantic HTML?

Semantic HTML provide structure and context as to what are in between certain tags. Unlike the <div></div> element, semantic HTML gives our HTML code a much more sturctured and clear outline of the content on the page. They also can make a webpage more accessible to mobile users and people with disabilities. There are many different types of semantic HTML elements that we can use.
